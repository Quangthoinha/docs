# Agent Frameworks

Đối với các bài toán phức tạp đòi hỏi code tùy chỉnh cao, Frameworks là lựa chọn bắt buộc.

| Framework | Ngôn ngữ | Đặc điểm chính |
| :--- | :--- | :--- |
| **LangChain** | Python/JS | Tiêu chuẩn ngành, hệ sinh thái khổng lồ, nhưng đôi khi cồng kềnh. |
| **LangGraph** | Python | Tập trung vào việc xây dựng stateful, multi-actor applications (Multi-Agent). Kiểm soát luồng tốt hơn LangChain thuần. |
| **Autogen** | Python | Của Microsoft. Mạnh về việc tạo ra nhiều agent nói chuyện với nhau để giải quyết vấn đề. |
| **CrewAI** | Python | Layer trên của LangChain, giúp định nghĩa Role và Task cho Agent dễ dàng hơn. |

## So sánh Autogen vs LangGraph
