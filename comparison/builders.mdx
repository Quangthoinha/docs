## 2. Deep Dive: Tool vs Product

<AccordionGroup>
  <Accordion title="Tại sao Dify là 'Product' tốt nhất?">
    Dify không chỉ cho bạn kéo thả luồng (Flow). Nó cung cấp cả một hệ thống backend hoàn chỉnh:
    *   **RAG Engine**: Upload PDF, Dify tự chia nhỏ (Chunking) và lưu vào Vector DB (Weaviate tích hợp sẵn).
    *   **API Management**: Tự sinh API Key để bạn đưa vào App Mobile/Web của mình.
    *   **Monitoring**: Xem log chat, đánh giá câu trả lời của AI.
  </Accordion>

  <Accordion title="Khi nào nên dùng Flowise/LangFlow?">
    Khi bạn muốn **nhúng sâu (Embed)**.
    Flowise và LangFlow xuất ra code định dạng JSON hoặc Python/JS. Bạn có thể nhúng thư viện của họ vào code backend NodeJS/Python của riêng bạn. Dify thường chạy như một service độc lập.
  </Accordion>
</AccordionGroup>

<Card title="Phán quyết (Verdict)" icon="gavel">
  *   **Doanh nghiệp (SME/Corp)**: Chọn **Dify** vì có sẵn phân quyền (Workspace), RAG pipeline ổn định và hỗ trợ tiếng Việt tốt.
  *   **Developer/Indie Hacker**: Chọn **Flowise** hoặc **LangFlow** để prototype nhanh và nhúng vào ứng dụng React/NextJS.
</Card>
