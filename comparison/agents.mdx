# Multi-Agent Orchestration (Chi tiết)

Khi một Agent đơn lẻ không đủ, chúng ta cần một đội ngũ. Dưới đây là thông số thực tế.

## 1. Bảng so sánh "Official Specs"

| Thông số | AgentGPT | CrewAI | LangGraph |
| :--- | :--- | :--- | :--- |
| **Giấy phép (License)** | **MIT** (Open Source). [[GitHub]](https://github.com/reworkd/AgentGPT) | **MIT** (Core framework). Enterprise features riêng. [[GitHub]](https://github.com/joaomdmoura/crewAI) | **MIT** (Core library). [[GitHub]](https://github.com/langchain-ai/langgraph) |
| **Pricing (Cloud)** | **$40/tháng** (Pro). Free tier giới hạn daily loops. [[Pricing]](https://agentgpt.reworkd.ai/) | **Enterprise Only**. Core là thư viện Python miễn phí. [[Site]](https://www.crewai.com/) | **Free** (Self-hosted). LangSmith (Cloud) giá từ **$0.001/run**. [[Pricing]](https://www.langchain.com/pricing) |
| **Mô hình** | **Autonomous Loops** on Browser | **Role-Playing Framework** (Code) | **State Machine / Graph** (Code) |

<Warning>
  **Giới hạn của AgentGPT (Pro Plan):**
  Ngay cả gói Pro ($40) cũng có giới hạn **30 agents/ngày** và **25 loops/lần chạy**. Nó phù hợp để Demo hơn là chạy tác vụ Production liên tục.
</Warning>

## 2. So sánh tư duy (Mental Models)

<CardGroup cols={3}>
  <Card title="AgentGPT" icon="globe">
    **Goal-based**
    "Tôi muốn làm affiliate marketing". Agent tự chia nhỏ task, tự chạy, tự Google. Ít quyền kiểm soát.
  </Card>
  <Card title="CrewAI" icon="users">
    **Role-based**
    "Tuyển 1 Researcher, 1 Writer". Bạn code định nghĩa vai trò của từng Agent, chúng nói chuyện với nhau bằng tiếng Anh (hoặc file cấu hình).
  </Card>
  <Card title="LangGraph" icon="diagram-project">
    **Graph-based**
    "Nếu A xong thì qua B, nếu lỗi thì quay lại A". Kiểm soát luồng đi cực chặt chẽ bằng Code (Graph theory). Ổn định nhất.
  </Card>
</CardGroup>

## 3. Phán quyết (Verdict)

*   **Production**: Bắt buộc chọn **LangGraph**. Sự kiểm soát trạng thái (State) và khả năng "Time Travel" (quay lại bước trước khi lỗi) là vô giá.
*   **MVP / PoC**: Chọn **CrewAI**. Dễ viết, code giống văn nói, setup nhanh một team agent để test ý tưởng.
*   **Nghịch ngợm / Demo**: Chọn **AgentGPT**. Chạy ngay trên web, không cần cài đặt.
